Get Started

Install

brew brew install k2n2-studio/tap/cortex
npx npx @cortex-memory/cli@latest init
curl curl -fsSL https://cortex.sh/install.sh | sh

How it works

1

Install & init

One command installs the CLI, starts the background daemon, and wires into Claude Code via MCP.

2

Claude learns as you work

Every decision, context, and preference is captured automatically into a local SQLite database.

3

Memory persists across sessions

When you start a new session, Claude recalls what it learned before. No re-explaining.

4

Sync across machines

Optional Turso sync keeps your memory consistent across laptops, desktops, and CI.
