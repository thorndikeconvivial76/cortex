# Changelog

All notable changes to the Cortex VS Code extension will be documented in this file.

## [1.0.0] — 2026-03-22

### Added

- Save memory from editor selection via command palette or right-click context menu
- Search memories with full-text search from within VS Code
- Hover peek to view related memories inline
- Status bar indicator showing daemon connection status and memory count
- SSE-powered session summary notifications
- Configurable daemon port and auto-start settings
- Keybindings: `Cmd+Shift+M` (save), `Cmd+Shift+F9` (search)
